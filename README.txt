== Installation ==
1. Download and unzip the plugin
2. Upload the entire "simple-poll-for-wp" directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to "Simple Poll" menu and configure your settings
